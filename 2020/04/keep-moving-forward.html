Keep Moving Forward

Dan | Sat 04 April 2020

I would probably rescind my post from a few days ago about how 'easy' it was to get this done and customize everything.

The initial lift was easy, getting the theme installed and following the documentation. The customization I wanted to do and matching things up to the demo of the Attila theme the way I wanted to. Being newer to Python still it was a good project to get exposure to a lot of libraries outside those I usually use like Pandas and SQLAlchemy.

The data source for the NY Covid data is no longer available for web scraping as the state has created a Tableau data visualization for tracking the outbreak. Obviously, this is great as the data being more accessible is better for everyone but it does put an end to the side project I had been running tracking the New York State data.

Ultimately I am glad that they have put that out there and for the experience gained in doing the web scraping.
